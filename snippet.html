<p>
Want to search the web?
Try <a href="https://google.com" id="the-best-link" class="banana"  target="_blank">Google</a>!
</p>

The tags in the snippet are paragraph tags <p> </p> and anchor tags <a> </a>.

Click on this link to learn about tags and this link to see some examples.

<a> </a> is a child of <p> </p>.

<p> is the opening tag for the paragraph and </p> is the closing tag for the paragraph.

<a> is the opening tag for the anchor and </a> is the closing tag for the anchor.

The attribute names on the <a> tag are href, id, class, and target.

The value for the attribute href is "https://google.com", the value for the attribute id is "the-best-link", the value for the attribute class is "banana", and the value for the attribute target is "_blank".

The href attribute and value "https://google.com" provide the "hyperlink reference to a linked resource Google."

The id attribute and value "the-best-link" help identifying the element.

The class attribute and value "banana" classifies the element under the class "banana".

The target attribute and value "_blank" indicates where the linked resource will open, and in this case, it will open in a new tab.
